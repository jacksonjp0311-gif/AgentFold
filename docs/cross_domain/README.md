# docs/cross_domain

## Echo Location

This folder stores bounded cross-domain analysis used to evolve AgentFold.

## Files

- `alphafold_genome_rna_agentfold_analysis.md` - source-bound analysis from AlphaFold, DNA/RNA expression, and RNA folding into AgentFold software architecture.
- `cross_domain_mapping.json` - machine-readable mapping of biological inspiration to software-only AgentFold surfaces.

## Boundary

These files use biology and molecular modeling as inspiration. They do not claim biological equivalence or wetlab validity.
